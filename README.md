# Игорь Сбитнев — backend-разработчик

Одностраничный сайт-визитка для backend-разработчика. Современный, минималистичный дизайн с автоматической сменой темы (светлая/темная) в зависимости от настроек устройства.

## Особенности

- Адаптивный дизайн
- Автоматическая светлая/темная тема (следит за системой)
- Неоновые акценты (зеленый и красный)
- Чистая типографика (Inter)
- Плавные анимации и hover-эффекты
- Кликабельные карточки проектов
